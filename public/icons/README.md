# Icons Directory

Place the following PWA icon files here:

- `icon-192.png` — 192×192 px icon (for Android home screen, PWA)
- `icon-512.png` — 512×512 px icon (for splash screen, app store)

Both icons should have a purple (#8b5cf6) background with a 🎓 graduation cap emoji or your logo.

You can generate these icons using:
- [Favicon.io](https://favicon.io/emoji-favicons/graduation-cap/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [PWABuilder](https://www.pwabuilder.com/imageGenerator)

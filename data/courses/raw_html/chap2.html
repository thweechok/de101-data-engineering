อะไรคือ CSS?

CSS ย่อมาจาก Cascading Style Sheets เป็นภาษาที่ใช้ในการกำหนดรูปแบบการแสดงผลของ HTML ไม่ว่าจะเป็น สี ขนาด เลย์เอาต์ หรือแม้กระทั่งแอนิเมชัน คำว่า "Cascading" หมายถึงการไหลตกทอดของสไตล์จากระดับบนสู่ระดับล่าง (ลำดับความสำคัญ)

การเชื่อมต่อ CSS กับ HTML (How to add CSS)

มี 3 วิธีหลักในการใส่ CSS แต่ในระดับมืออาชีพ เราจะใช้ External CSS เป็นหลัก:

  • Inline CSS: ใส่สไตล์ตรงๆ ใน tag HTML ผ่าน attribute style="..." (ไม่แนะนำเพราะโค้ดจะรก)
  • Internal CSS: เขียนโค้ด CSS ไว้ในแท็ก <style> ภายใน <head> ของ HTML
  • External CSS: เขียน CSS แยกไฟล์ (เช่น style.css) แล้วเชื่อมด้วย <link rel="stylesheet" href="style.css"> (แนะนำที่สุด โค้ดสะอาด นำไปใช้ซ้ำได้)

Selectors: การเลือกองค์ประกอบ (Targeting Elements)

เพื่อให้ CSS รู้ว่าเราต้องการปรับแต่งส่วนไหน เราต้องใช้ Selectors ที่ถูกต้อง:

  1. Element Selector: เลือกตามชื่อ tag เช่น p { color: red; } จะทำให้ทุกย่อหน้าเป็นสีแดง
  2. Class Selector (ใช้บ่อยสุด): เลือกตาม attribute class นำหน้าด้วยจุด (dot) เช่น .btn { border-radius: 5px; } (ใช้ซ้ำได้หลายที่)
  3. ID Selector: เลือกตาม attribute id นำหน้าด้วยสี่เหลี่ยม (#) เช่น #header { background: black; } (1 หน้าควรมี id นั้นแค่ตัวเดียว)
  4. Attribute Selector: เลือกจากคุณสมบัติ เช่น input[type="text"] { width: 100%; }

Specificity: กฎความสำคัญของ CSS

เมื่อมีสไตล์ซ้ำซ้อนกันชี้ไปที่องค์ประกอบเดียวกัน CSS จะตัดสินใจเลือกสไตล์ที่มี Specificity Score สูงกว่ามาใช้งาน กฎพื้นฐานคือ: Inline Style > ID > Class > Element

หากคะแนนเท่ากัน CSS จะเลือกกฏที่ เขียนไว้ล่างสุด (ลำดับหลังสุด) มาใช้ ตัวอย่างเช่น ถ้าคุณมี .title { color: blue; } และ .title { color: red; } ข้อความจะกลายเป็นสีแดง

Colors และ Typography (สีและตัวอักษร)

การจัดการสีใน CSS สามารถทำได้หลายรูปแบบ:

  • Keyword: เช่น red, blue, transparent
  • HEX: รหัสฐาน 16 เช่น #FF0000, #000000
  • RGB / RGBA: เช่น rgb(255, 0, 0) และ rgba เพิ่ม alpha channel สำหรับความโปร่งใส rgba(0, 0, 0, 0.5) (โปร่งใส 50%)
  • HSL / HSLA: กำหนดจาก Hue, Saturation, Lightness

สำหรับ Typography (ตัวอักษร) Properties ที่สำคัญคือ:

body {
    font-family: 'Inter', sans-serif; /* ฟอนต์หลักและฟอนต์สำรอง */
    font-size: 16px; /* ขนาดตัวอักษร */
    font-weight: 700; /* ความหนา (400=normal, 700=bold) */
    line-height: 1.5; /* ระยะห่างระหว่างบรรทัด */
    color: #333333; /* สีตัวอักษร */
    text-align: center; /* การจัดตำแหน่ง */
}
💡 Pro Tip: อย่าใช้ !important ใน CSS หากไม่จำเป็นจริงๆ เพราะมันจะทำลายระบบ Specificity และทำให้โค้ดของคุณพัง แก้ไข (Debug) ยากมากในอนาคต! ให้แก้ปัญหาโดยจัดลำดับความสำคัญของ Class ให้ถูกต้องแทน

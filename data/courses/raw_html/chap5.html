การปฏิวัติวงการ Layout: CSS Flexbox

ก่อนหน้าที่โลกจะมี Flexbox (Flexible Box Module) นักพัฒนาต้องทนทุกข์กับการใช้ Float, Table หรือ Inline-block ในการจัด Layout ซึ่งซับซ้อนและพังได้ง่าย Flexbox ถูกออกแบบมาเพื่อแก้ปัญหานี้ โดยช่วยให้เราจัดเรียงกล่อง (Elements) ใน "1 มิติ" (แถวเดียว หรือ คอลัมน์เดียว) ได้อย่างง่ายดายและยืดหยุ่น

Flex Container vs Flex Items

ในการใช้ Flexbox คุณต้องเริ่มจากการกำหนดให้กล่องแม่ (Parent) กลายเป็น Flex Container ด้วยคำสั่ง display: flex; (หรือ inline-flex) ทันทีที่คุณทำเช่นนั้น กล่องลูกที่อยู่ข้างใน (Children) จะกลายเป็น Flex Items และเริ่มประพฤติตามกฎของ Flexbox ทันที

Properties สำหรับ Flex Container (กล่องแม่)

  1. flex-direction: กำหนดทิศทางหลัก (Main Axis) ว่าจะเรียงของอย่างไร
    • row (ค่าเริ่มต้น): เรียงแนวนอน จากซ้ายไปขวา
    • column: เรียงแนวตั้ง จากบนลงล่าง
    • row-reverse / column-reverse: เรียงกลับด้าน
  2. justify-content: การจัดแนวของ Items ตามแกนหลัก (Main Axis)
    • flex-start: ชิดจุดเริ่มต้น (ซ้าย หรือ บน)
    • center: จัดกึ่งกลาง
    • space-between: ดัน Items ไปชิดขอบสองฝั่ง และแบ่งช่องว่างตรงกลางให้เท่าๆ กัน (ใช้บ่อยในการทำ Navbar)
    • space-around / space-evenly: แบ่งช่องว่างรอบๆ Items แต่ละตัว
  3. align-items: การจัดแนวของ Items ตามแกนรอง (Cross Axis)
    • stretch (ค่าเริ่มต้น): ยืดให้เต็มความสูง/กว้าง ของ Container
    • center: จัดกึ่งกลางในแนวแกนรอง (มีประโยชน์มากในการจัดให้อยู่ตรงกลางเป๊ะๆ)
    • flex-start / flex-end: ชิดจุดเริ่มต้น หรือ จุดสิ้นสุดของแกนรอง
  4. flex-wrap: หากพื้นที่ไม่พอ จะให้ Items บีบตัวเองหรือปัดตกบรรทัดใหม่
    • nowrap (ค่าเริ่มต้น): บีบให้พอในบรรทัดเดียว
    • wrap: ถ้าที่เต็ม ให้ปัดตกไปบรรทัดใหม่

Properties สำหรับ Flex Items (กล่องลูก)

คุณยังสามารถควบคุมลูกแต่ละตัวแยกกันได้ด้วย:

  • flex-grow: อัตราการ "ขยายตัว" เพื่อเติมเต็มพื้นที่ที่เหลืออยู่ (เช่น flex-grow: 1; หมายถึง ให้ยืดกินพื้นที่ที่เหลือทั้งหมด)
  • flex-shrink: อัตราการ "หดตัว" เมื่อพื้นที่ไม่พอ
  • flex-basis: ขนาดเริ่มต้นของ Item ก่อนที่ grow หรือ shrink จะทำงาน (คล้ายๆ width)
  • order: เปลี่ยนลำดับการแสดงผลของ Item โดยไม่ต้องไปสลับใน HTML (ค่าเริ่มต้นคือ 0)
💡 Pro Tip: การจัด Element ให้อยู่กึ่งกลางหน้าจอแบบ Perfect Center (ทั้งแกน X และ Y) ที่ง่ายที่สุดในโลกคือการใช้:
display: flex; justify-content: center; align-items: center; ในกล่องแม่!

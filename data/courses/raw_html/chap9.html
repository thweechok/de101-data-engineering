เพิ่มชีวิตชีวาให้หน้าเว็บด้วย Animations

เว็บไซต์ที่ดีไม่ควรนิ่งเป็นหิน การเพิ่มความเคลื่อนไหว (Motion) เล็กๆ น้อยๆ เมื่อผู้ใช้ปฏิสัมพันธ์ (เช่น นำเมาส์ไปชี้) จะช่วยเพิ่ม User Experience (UX) ได้อย่างมหาศาล และทำให้เว็บดูมีความเป็นมืออาชีพ CSS มีเครื่องมือหลัก 2 อย่างในการทำสิ่งนี้: Transitions และ Animations

CSS Transitions (การเปลี่ยนแปลงแบบสมูท)

transition ใช้สำหรับเปลี่ยนสถานะ (State) ขององค์ประกอบจากแบบหนึ่งเป็นอีกแบบหนึ่งอย่างนุ่มนวล (มักใช้ร่วมกับ Pseudo-classes เช่น :hover หรือ :focus) แทนที่จะเปลี่ยนสีปุ่มแบบกระพริบฉับพลัน เราทำให้มันค่อยๆ เฟดสีได้

.btn {
    background-color: #3498db;
    /* transition: [property] [duration] [timing-function] [delay]; */
    transition: background-color 0.3s ease-in-out;
}

.btn:hover {
    background-color: #2980b9; /* เมื่อ hover สีจะเปลี่ยนเป็นสีนี้ โดยใช้เวลา 0.3 วิ */
}

Transform (การบิดเบือนรูปทรงและตำแหน่ง)

property transform ไม่ใช่แอนิเมชันในตัวมันเอง แต่มันคือการปรับเปลี่ยนรูปทรง (ขยับ, หมุน, ย่อ/ขยาย) ซึ่งมักนำมาใช้คู่กับ transition เพื่อสร้างเอฟเฟกต์ที่ลื่นไหล

  • translate(x, y): ขยับตำแหน่ง (นิยมใช้แทน top/left เพราะเบราว์เซอร์จะคำนวณผ่าน GPU ทำให้ลื่นไหลกว่ามาก ไม่กระตุก)
  • scale(n): ย่อขยายขนาด (1 คือขนาดปกติ, 1.1 คือใหญ่ขึ้น 10%)
  • rotate(deg): หมุนเป็นองศา (เช่น rotate(45deg))
.card {
    transition: transform 0.2s ease;
}
.card:hover {
    transform: translateY(-10px) scale(1.02); /* ขยับขึ้น 10px และขยาย 2% ตอน hover */
}

CSS Keyframes Animation (แอนิเมชันแบบควบคุมเต็มขั้น)

หากคุณต้องการให้แอนิเมชันเล่นเองตั้งแต่โหลดหน้าเว็บ เล่นวนลูป หรือมีหลายๆ สเต็ป (ไม่ใช่แค่ A ไป B) คุณต้องใช้ @keyframes

เราต้องกำหนดชุดของเฟรมก่อน:

@keyframes pulse-glow {
    0% { transform: scale(1); box-shadow: 0 0 0 rgba(255,0,0,0); }
    50% { transform: scale(1.1); box-shadow: 0 0 20px rgba(255,0,0,0.5); }
    100% { transform: scale(1); box-shadow: 0 0 0 rgba(255,0,0,0); }
}

จากนั้นนำชื่อ pulse-glow ไปเรียกใช้ด้วย property animation:

.alert-icon {
    /* animation: [name] [duration] [timing] [delay] [iteration-count]; */
    animation: pulse-glow 2s ease-in-out infinite; /* เล่นลูปไม่สิ้นสุด (infinite) */
}
💡 Pro Tip เพื่อประสิทธิภาพ (Performance): กฎทองของการทำแอนิเมชันคือ ให้ Animate เฉพาะ `transform` และ `opacity` เท่านั้น! การทำแอนิเมชันกับ property อย่าง width, height, หรือ margin จะทำให้เบราว์เซอร์ต้องคำนวณ Layout ใหม่ทั้งหมด (Reflow/Repaint) ทุกๆ เฟรม ซึ่งจะทำให้คอมพิวเตอร์และมือถือของผู้ใช้กระตุกและกินแบตเตอรี่มหาศาล!

CSS Custom Properties (ตัวแปรใน CSS)

ในอดีต หากเว็บไซต์ของคุณมี "สีหลัก (Primary Color)" ที่ใช้ในปุ่ม 50 แห่ง, ข้อความ 20 แห่ง, และขอบ 30 แห่ง เมื่อดีไซเนอร์ขอเปลี่ยนสี คุณต้องตามหาแก้โค้ดถึง 100 จุด (หาและแทนที่) ปัญหานี้ได้รับการแก้ไขด้วย CSS Variables (หรือชื่ออย่างเป็นทางการคือ Custom Properties)

การสร้างและเรียกใช้งาน (Syntax)

ตัวแปรใน CSS จะขึ้นต้นด้วยขีดสองอัน (--) เสมอ เรามักจะประกาศตัวแปรไว้ใน :root (ซึ่งเทียบเท่ากับแท็ก <html>) เพื่อให้สามารถเรียกใช้ได้แบบ Global (ทุกที่ในเว็บไซต์)

:root {
    --primary-color: #3498db;
    --secondary-color: #2ecc71;
    --text-dark: #333333;
    --bg-light: #f4f4f4;
    --spacing-md: 16px;
    --radius: 8px;
}

เมื่อต้องการนำไปใช้ เราจะใช้ฟังก์ชัน var() และใส่ชื่อตัวแปรลงไป:

.button {
    background-color: var(--primary-color);
    color: white;
    padding: var(--spacing-md);
    border-radius: var(--radius);
}

พลังของการเขียนทับ (Cascading Variables)

แม้จะชื่อว่า "ตัวแปร Global" แต่เราสามารถเปลี่ยนค่าของตัวแปรในขอบเขต (Scope) เฉพาะเจาะจงได้ ซึ่งเป็นจุดที่ CSS Variables ทรงพลังกว่าตัวแปรใน Sass (SCSS) ตัวอย่างเช่น คุณสามารถเปลี่ยนค่าตัวแปรเฉพาะใน Section หนึ่งได้:

.card.highlight {
    /* เปลี่ยนสีหลักเฉพาะข้างใน card ที่มีคลาส highlight */
    --primary-color: #e74c3c; 
}
/* ปุ่มที่อยู่ใน card.highlight จะกลายเป็นสีแดงอัตโนมัติ โดยไม่ต้องเขียนคลาสปุ่มใหม่ */

การทำ Dark Mode อย่างง่ายดาย

Use Case ที่ทำให้ CSS Variables เปล่งประกายที่สุดคือการทำ ธีม (Theming) และ Dark Mode แทนที่คุณจะเขียน CSS ใหม่ทั้งหมดสำหรับ Dark Mode คุณแค่เพียง "เปลี่ยนค่าของตัวแปร" เมื่อหน้าจอหรือคลาสเปลี่ยนไป

/* สีสำหรับ Light Mode (ค่าเริ่มต้น) */
:root {
    --bg-color: #ffffff;
    --text-color: #121212;
    --border-color: #cccccc;
}

/* สีสำหรับ Dark Mode */
@media (prefers-color-scheme: dark) {
    :root {
        --bg-color: #121212;
        --text-color: #ffffff;
        --border-color: #333333;
    }
}

/* นำไปใช้ในโค้ด (ไม่ต้องสนใจว่าเป็นโหมดอะไร) */
body {
    background-color: var(--bg-color);
    color: var(--text-color);
}

ในตัวอย่างด้านบน เราใช้ Media Query @media (prefers-color-scheme: dark) ซึ่งจะเช็คว่าผู้ใช้ตั้งค่า OS ของโทรศัพท์หรือคอมพิวเตอร์เป็น Dark Mode หรือไม่ ถ้าใช่ เว็บคุณจะกลายเป็น Dark Mode อัตโนมัติทันที!

💡 Pro Tip: ฟังก์ชัน var() รับพารามิเตอร์ตัวที่ 2 เพื่อทำเป็นค่า Fallback (ค่าเผื่อสำรอง) ได้ เช่น color: var(--theme-color, #000); หมายความว่า ถ้ายังไม่ได้ประกาศ --theme-color ให้ใช้สีดำ (#000) ไปก่อน ป้องกันเว็บพัง!

1. สู่ยุคใหม่ของการจัด Layout ด้วย Flexbox

ก่อนยุคของการมี Flexbox การจัดเรียงเลย์เอาต์ที่ซับซ้อน (เช่น การจัดคอนเทนต์ให้อยู่กึ่งกลางหน้าจอพอดี, การแบ่งพื้นที่ให้ความกว้างเท่าๆ กัน, หรือการให้กล่องมีความสูงเท่ากันอัตโนมัติ) เป็นฝันร้ายของ Web Developer ที่ต้องพึ่งพาวิธีการแฮ็ก (Hack) แบบแปลกๆ เช่น การใช้ float ร่วมกับ clearfix หรือการใช้ table จำลอง

แต่การมาถึงของโมดูล Flexbox (Flexible Box Layout) ได้เปลี่ยนวงการนี้ไปตลอดกาล Flexbox ถูกออกแบบมาเพื่อการจัดเลย์เอาต์แบบ 1 มิติ (1-Dimensional) นั่นคือจัดการองค์ประกอบใน แนวนอน (Row) หรือ แนวตั้ง (Column) อย่างใดอย่างหนึ่งในแต่ละครั้ง

2. กฎพื้นฐาน: Flex Container และ Flex Items

การจะใช้เวทมนตร์นี้ คุณต้องเปลี่ยนกล่องแม่ (Parent Element) ให้กลายเป็นเวทีประลองเสียก่อน ด้วยการใส่คำสั่ง display: flex; (หรือ inline-flex) เมื่อแม่กลายเป็น Flex Container แล้ว ลูกๆ ทุกตัวที่อยู่ข้างในระดับชั้นแรกสุด (Direct Children) จะถูกบังคับให้กลายร่างเป็น Flex Items โดยอัตโนมัติ

.container {
  display: flex;
}

โดยค่าเริ่มต้น เมื่อเราสั่ง display: flex ลูกๆ ทุกตัวจะวิ่งมาเรียงต่อกันเป็นแถวแนวนอนจากซ้ายไปขวาทันที โดยไม่สนว่าลูกจะเป็น tag block หรือ inline

3. การควบคุมแกนหลัก (Main Axis) และแกนรอง (Cross Axis)

ในโลกของ Flexbox เราจะไม่คุยกันเรื่อง บน-ล่าง-ซ้าย-ขวา แต่เราจะคุยกันเรื่อง Main Axis (แกนหลักที่ไอเท็มเรียงตัว) และ Cross Axis (แกนรองที่ตั้งฉากกับแกนหลัก) คุณสามารถพลิกทิศทางของแกนหลักได้ด้วย Property flex-direction

  • flex-direction: row; (ค่าเริ่มต้น) แกนหลักวิ่งแนวนอน จากซ้ายไปขวา
  • flex-direction: column; แกนหลักวิ่งแนวตั้ง จากบนลงล่าง
  • row-reverse และ column-reverse (กลับด้านจากขวามาซ้าย หรือล่างขึ้นบน)

4. Properties สำหรับ Flex Container (สั่งงานที่แม่)

เราสามารถจัดระเบียบลูกๆ ได้ด้วยคำสั่งเหล่านี้ที่กล่องแม่:

  1. justify-content (จัดแนวบนแกนหลัก - Main Axis):
    • flex-start: ชิดจุดเริ่มต้นของแกนหลัก (ซ้ายสุด)
    • flex-end: ไปกองรวมกันที่จุดสิ้นสุด (ขวาสุด)
    • center: กระจุกตัวอยู่กึ่งกลาง
    • space-between: ตัวแรกชิดซ้ายสุด ตัวสุดท้ายชิดขวาสุด ที่เหลือแบ่งช่องว่างตรงกลางเท่าๆ กัน (นิยมใช้ทำ Navbar)
    • space-around / space-evenly: กระจายช่องว่างรอบๆ ไอเท็ม
  2. align-items (จัดแนวบนแกนรอง - Cross Axis):
    • stretch (ค่าเริ่มต้น): ยืดความสูงให้เต็มคอนเทนเนอร์แม่โดยอัตโนมัติ
    • center: อยู่กึ่งกลางแนวแกนรอง (ใช้คู่กับ justify-content: center เพื่อทำให้อยู่ตรงกลางจอเป๊ะๆ)
  3. flex-wrap (พฤติกรรมการปัดบรรทัด):

    โดยปกติ Flex Items จะพยายามเบียดบี้กันให้อยู่ในบรรทัดเดียวเสมอ แม้จะต้องหดตัวจนเสียรูปทรง ถ้าคุณอยากให้มันปัดตกไปขึ้นบรรทัดใหม่เมื่อพื้นที่ความกว้างหมด ให้ใช้ flex-wrap: wrap;

5. Properties สำหรับ Flex Items (อิสระที่ฝั่งลูก)

ความยืดหยุ่น (Flexible) ที่แท้จริงอยู่ที่ลูกๆ เราสามารถกำหนดให้ลูกแต่ละตัวยืดหรือหดเพื่อแย่งชิงพื้นที่ว่างได้:

  • flex-grow: อัตราการขยายตัวเพื่อกินพื้นที่ว่าง (ค่าเริ่มต้นคือ 0 คือไม่ขยายเลย) ถ้าใส่ 1 มันจะยืดไปกินพื้นที่ว่างที่เหลือในบรรทัดจนหมด
  • flex-shrink: อัตราการหดตัวเมื่อพื้นที่ไม่พอ (ค่าเริ่มต้นคือ 1 คือยอมหด) ถ้าใส่ 0 คือห้ามหดเด็ดขาด
  • flex-basis: ขนาดตั้งต้น (คล้ายๆ width แต่ฉลาดกว่าเพราะอ้างอิงตามทิศทางของแกน) ก่อนที่จะทำการยืดหรือหด
💡 Pro Tip: การจัดกึ่งกลางที่สมบูรณ์แบบ (Perfect Centering) นี่คือโค้ดสุดคลาสสิกที่แก้ปัญหาปวดหัวระดับตำนานของนักพัฒนาเว็บ หากคุณต้องการให้ข้อความ หรือกล่องใดๆ ลอยอยู่ตรงกลางหน้าจอเป๊ะๆ ทั้งแนวนอนและแนวตั้ง เพียงแค่ใช้ 3 บรรทัดนี้ที่กล่องแม่: 
.parent {
  display: flex; 
  justify-content: center; /* กึ่งกลางแนวนอน */
  align-items: center; /* กึ่งกลางแนวตั้ง */
  height: 100vh; /* ให้กล่องแม่สูงเต็มจอ */
}
